An affiliate link management system for your blog or site, featuring link rotation with an easy-to-use interface.

== Description ==
Link Optimizer Lite is a Wordpress plugin that provides a comprehensive link-management system for your blog or site.
With Link Optimizer, you can:

* Create groups of links and rotate through them
* Manage or change your affiliate links from one place
* Create link-rotating shortlinks for use with ad networks and social media


With just three easy steps, you can start optimizing your blog today:

* Create a group for your links
* Add as many links as you like to the set, varying the link text
* In your posts, pages, and text widgets, insert the special tag [linkoptimizer:X], which you can find in the admin page.

That's it! For more information, check out the [User Guide](http://refinry.com/affiliate-link-optimizer/user-guide/).


You can upgrade at any time to [Link Optimizer](http://refinry.com/affiliate-link-optimizer/),
which also provides hit/click tracking and automatic conversion optimization.


== Installation ==

1. Unzip `link-rotator-lite-<version>.zip` into the `/wp-content/plugins/` directory, or use the `Add Plugin` link on the plugin management page.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Navigate to the `Link Optimizer` submenu under the `Plugins` menu, click "Edit", and set up your links
1. Copy and paste the `[linkrotator:X]` tag into your posts, pages, or text widgets where you want your link to appear.
